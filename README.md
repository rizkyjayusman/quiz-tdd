# Build and Run


```
1. Clone the Project
   $ git clone git@github.com:rizkyjayusman/quiz-tdd.git
   $ cd quiz-tdd

2. Build and Package the Project
   $ mvn -e clean package

3. Run the Project
   $ java -jar /quiz-tdd-project-path/app.jar
```
